Code Beautifier Açıklamaları
----------------------------
* Kod güzelleştirme yapmak için "code_beautifier.bat" scriptinin çalıştırılması yeterlidir. 

* Script çalıştığında proje "Source" dizinindeki "*.c", "*.h" ve "*.cpp" uzantılı dosyalar 
  uncrustify.exe uygulaması ve uncrustify.cfg konfigürasyon dosyası kullanılarak güzelleştirme 
  işlemine tabi tutulmaktadır.
